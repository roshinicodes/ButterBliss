# ButterBliss

A simple, full-stack-feel bakery ordering web app built with HTML, CSS, and JavaScript — browse the menu, add items to your cart, and see your order total calculated in real time, all tied to a personal session.

**📖 Project Description**


ButterBliss is a front-end bakery ordering website where users log in with their name, mobile number, and address, browse a menu of baked goods, and build an order using an interactive quantity selector. Cart data is stored using the browser's localStorage, so selections persist across pages without needing a backend. This project was built to learn core concepts such as DOM manipulation, event handling, client-side session management, and data persistence with localStorage.

**Features**

🔐 Simple Login – Sign in using your name, mobile number, and address. No password required.

🍞 Interactive Menu – Browse bakery items (croissants, cakes, donuts, and more) displayed in a responsive card grid.

➕ Add to Cart – Adjust item quantities with +/- buttons, with live updates saved to your cart.


🧾 Cart Summary – View all selected items with individual prices, quantities, and a calculated grand total.


🚪 Session Handling – Logged-in state is checked on the menu page, redirecting unauthenticated users back to login; logout clears the session.


🎨 Custom Themed UI – Warm bakery-inspired design with Google Fonts, Font Awesome icons, and an embedded Google Maps location for the store.
